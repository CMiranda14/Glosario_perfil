FÓRMULARIO DE TIQUETAS (HTML) Y PROPIEDADES (CSS)



TIPOS DE ETIQUETAS


En HTML y CSS tenemos diferentes tipos de etiquetas cada uno con un propósito en
particular una etiqueta se compone de dos maneras.


ESTRUCTURA DE ETIQUETAS
Etiquetas que solo tienen apertura Etiqueta que tiene una apertura y un sierre
Este tipo de etiquetas conmúnmente se utilizan para indicar una dirección
u origen se componen por argumento u opciones tales como src href. 		Este tipo de etiquetas conmúnmente se utiilizan para encerrar un contenido
tales como texto, img,etc.


Lista de etiquetas dirigidas al texto:


Etiqueta Función
🏀< p ></p> Se utiliza para distribuir un texto en parrafo.
🏀<h1></h1> Todas las etiquetas (h1,h2,h3,h4,h5,h6) nos sirven para colocar un título.


Lista de etiquetas dirigidas a objetos:


Etiqueta Función
🏀< img> se utiliza para insertar una imagen.
🏀<header > Puede contener algunos elementos de encabezado, así
como también un logo, fórmulario de búsqueda, un nombre de usuario, etc.


Lista de etiquetas para definir el documento:


Etiqueta Función
🏀<!DOCTYPE> Define el tipo de documento.
🏀<title></title> El contenido dentro de ella será un título.
🏀<body></body> Representa el contenido del documento.
🏀<head></head> Contiene información que describe el documento de la página web.


Etiquetas para una Lista:


Etiqueta Función
🏀<ol></ol> Hace referencia a una lista ordenada.
🏀<li></li> Son los elementos de la lista.
🏀<ul></ul> Sirve para realizar una lista desordenada.


Etiquetas para realizar una Tabla:


Etiqueta Función
🏀<table></table> Nos sirve para poder crear una tabla.
🏀<tr></tr> Sirve para colocar filas.
🏀<td></td> Nos sirve para colocar columnas en la tabla.


Lista de etiquetas para poder colocar links:


Etiqueta Función
🏀<a></a> Sirve para colocar links de páginas, imágenes,
documentos, etc.
🏀Atributo href "<a href="link"></a> Nos sirve para colocar la dirrección de el link.


Lista de etiquetas para crear un Fórmulario:


Etiqueta Función
🏀<form></form> Sirve para poder colocar un fórmulario.
🏀<label></label> Sirve para poder colocar un título en el fórmulario.
🏀<input></input> Sirve para colocar una entrada de texto.
🏀Atributo "type" <input type="text"></type> Sirve para especificar que es lo que se debe colocar,
texto,gmail, etc.
🏀<button></button> Sirve para poder colocar un botón.


Lista de etiquetas dirigidas a Maquetación:


Etiqueta Función
🏀 <header></header> Sirve para colocar encabezados, títulos, etc.
🏀<nav></nav> Sirve para poder colocar todo el navegador.
🏀<section></section> Sirve para poder colocar elementos en secciones.
🏀<article></article> Representa una composición auto-contenida en un documento.
🏀<footer></footer> Representa un pie de página, para el contenido de sección mas cercano
o elemento raíz de sección.


Connexión html a css:


🏀Colocamos la etiqueta <link>, luego colocamos dentro de la etiqueta un atributo "rel" y dentro de ese rel entre "",
colocamos la palabra "stylesheet". Luego colocamos otro atributo junto a "rel", el cual es "href" y dentro de ese atributo
colocamos el nombre de nuestro css. Ejemplo:


<link rel="stylesheet"href="css/design.css">.


Lista de propiedades css dirigidas a texto:


Propiedad Función
🏀"color: #fffff;" Inserta un color al texto mediante el nombre del color, o en este caso
un código de color.
🏀font-size: 1px Específica el tamaño en que queremos que este nuestro texto.
🏀text-ident Desplazamiento de la primera línea del texto. Sus valores son longitud
y porcentaje.
🏀text-aling Alineamiento del texto. Sus valores son: left, right, center y justify.


Listado de propiedades dirigidas a objetos:


Propiedad Función
🏀whidth: 100px; Establece la achura de la visualización de la imagen.
🏀with: puede anchar la imagén.
🏀height Puede estirar la imagén.
🏀border coloca un borde al elemento escogido.
🏀border-radious Puede redondear el borde de las imagenes.
🏀background-color coloca un colo de fondo al elemento.


Lista de propiedades dirigidas para Tablas:


Propiedad Función
🏀width Define el ancho de un elemento, y le valor se pued escribir en pixeles, ems,
o porcentaje.
🏀heightEspecifíca la altura del área de contenido de un elemento.
🏀padding Es la distancia desde el borde de un elemento hasta su contenido.
🏀margin Es la distancia entre un elemento y otro (desde el borde del elemento hacia
afuera).
🏀border Define el borde de un elemento, su color, su estilo y grosor.
